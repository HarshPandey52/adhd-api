# ADHD Prediction API

EEG-based ADHD prediction via **Riemannian geometry + XGBoost**, served with **FastAPI**.

---

## Project Structure

```
adhd-api/
├── app/
│   ├── __init__.py
│   ├── main.py          ← FastAPI app + endpoints
│   ├── schema.py        ← Pydantic input/output models
│   └── predictor.py     ← Feature extraction + inference
├── model/               ← Generated by train_and_save_model.py
│   ├── xgb_model.json
│   ├── scaler.pkl
│   └── riemann_pipeline.pkl
├── train_and_save_model.py  ← Run once on your dataset
├── requirements.txt
├── Dockerfile
├── render.yaml
└── README.md
```

---

## Step 1 — Train & Save the Model

Run this **once** on your machine (with your dataset):

```bash
pip install -r requirements.txt
python train_and_save_model.py --data /path/to/your_dataset.csv
```

This creates `model/xgb_model.json`, `model/scaler.pkl`, `model/riemann_pipeline.pkl`.

---

## Step 2 — Run Locally

```bash
uvicorn app.main:app --reload
```

Visit:
- API docs → http://localhost:8000/docs
- Health check → http://localhost:8000/health

---

## Step 3 — Deploy

### Option A: Render (easiest)

1. Push this folder to a GitHub repo (include the `model/` folder).
2. Go to https://render.com → New → Web Service → connect your repo.
3. Render auto-detects the `Dockerfile`. Hit **Deploy**.
4. Your API will be live at `https://your-service.onrender.com`.

### Option B: Google Cloud Run

```bash
# Build and push image
gcloud builds submit --tag gcr.io/YOUR_PROJECT/adhd-api

# Deploy
gcloud run deploy adhd-api \
  --image gcr.io/YOUR_PROJECT/adhd-api \
  --platform managed \
  --allow-unauthenticated \
  --port 8000 \
  --memory 2Gi
```

> ⚠️ Set memory to at least **2Gi** — MNE and pyriemann are memory-hungry.

## API Reference

### `POST /predict`

**Request body:**
```json
{
  "eeg_data": [
    [fp1, fp2, f3, f4, c3, c4, p3, p4, o1, o2,
     f7, f8, t7, t8, p7, p8, fz, cz, pz],
    ...
  ]
}
```

- Shape: `[n_timepoints, 19]`
- Values: microvolts (µV)
- Minimum: 750 timepoints (3 seconds × 250 Hz)
- Channel order: `Fp1 Fp2 F3 F4 C3 C4 P3 P4 O1 O2 F7 F8 T7 T8 P7 P8 Fz Cz Pz`

**Response:**
```json
{
  "prediction": 1,
  "label": "ADHD",
  "confidence": 0.7834,
  "confidence_pct": "78.34%",
  "threshold_used": 0.45
}
```

## Connecting to Your Website

### JavaScript (fetch)
```javascript
const response = await fetch("https://your-api-url.com/predict", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ eeg_data: yourEegArray })
});
const result = await response.json();
console.log(result.label);        // "ADHD" or "Non-ADHD"
console.log(result.confidence_pct); // "78.34%"
```

### Python (requests)
```python
import requests
response = requests.post(
    "https://your-api-url.com/predict",
    json={"eeg_data": your_eeg_array.tolist()}
)
print(response.json())
```
